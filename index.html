Flex box

Свойство CSS, позволяющее выстраивать блоки в соответствии с сеткой на равном расстоянии друг от друга.

.whatever { display: flex; flex-wrap: wrap; justify-content: space-between; }

Scroll magic

Анимация поэтапно привязывается к скроллу и происходит постепенно в соответствии с продвижением пользователя по странице

let controller = new ScrollMagic.Controller()

new ScrollMagic.Scene({ triggerElement: train1Anim, triggerHook: "onEnter", duration: 2700 })
